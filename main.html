Skill Matcher Project

This repository provides a Streamlit-based internship skill matcher tool.

GitHub Pages Deployment

GitHub Pages can host this static landing page. Dynamic Streamlit app must run via a server externally, or via Render/Heroku.

To run locally:

  1. Create and activate virtual environment
  2. Install dependencies from skill_matcher/requirements.txt
  3. Run the app with: streamlit run skill_matcher/app.py
